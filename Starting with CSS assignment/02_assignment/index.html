According to BEM (Block Elements Modifier) convention,we should write class name of parent elements according to semantic tags or content,and their child elements class name is written like parent class name__child class name. Modifiers elements should be written as classname--modifiers class name Example: Parent class name (Block): heading. Child class name (Elements): heading__paragraph. Modifiers class name: btn--backgroundcolor.
